# Icons Folder

Place your custom icons and small graphics here.

## Recommended Icons:

- Service icons (AI, chatbot, analytics, etc.)
- Social media icons (if custom)
- UI icons (arrows, checkmarks, etc.)

## Format:
- Prefer SVG format for scalability
- Alternative: PNG with transparent background

## Naming Convention:
- `icon-service-ai.svg`
- `icon-social-twitter.svg`
- etc.

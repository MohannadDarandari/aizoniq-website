# Images Folder

Place your website images here:

## Recommended Images:

### Logo & Branding
- `logo.png` - Main logo (transparent background)
- `logo-white.png` - White version for dark backgrounds
- `favicon.png` - Browser favicon (32x32 or 64x64)
- `icon-192.png` - PWA icon (192x192)
- `icon-512.png` - PWA icon (512x512)

### Open Graph & Social Media
- `og-image.jpg` - Open Graph image (1200x630)
- `screenshot-desktop.png` - Desktop screenshot (1920x1080)
- `screenshot-mobile.png` - Mobile screenshot (750x1334)

### Portfolio & Projects
- `project-1.jpg`
- `project-2.jpg`
- `project-3.jpg`
- etc.

### Team Members
- `team-ceo.jpg`
- `team-cto.jpg`
- etc.

### Background & Decorative
- `hero-bg.jpg` - Hero section background
- `pattern.svg` - Background patterns

## Image Optimization Tips:

1. **Format:**
   - Use WebP for photos (best compression)
   - Use PNG for logos/icons with transparency
   - Use SVG for icons and simple graphics

2. **Size:**
   - Hero images: max 2000px width
   - Portfolio images: max 1500px width
   - Team photos: 400x400px
   - Thumbnails: 600x400px

3. **Compression:**
   - Use tools like TinyPNG or Squoosh
   - Aim for <200KB per image

4. **Naming:**
   - Use descriptive names: `hero-ai-technology.jpg`
   - Use lowercase and hyphens
   - Avoid spaces and special characters
